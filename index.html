Human & Crowd Detection Tool
2024

A real-time human detection and crowd monitoring system built with Python and OpenCV that tracks the number of people in frame and triggers visual and alert signals when crowd density exceeds a defined threshold.

  • Detects and counts humans in live video feed using HOG + SVM and DNN-based person detection pipelines with bounding-box annotation per individual.
  • Implements a 3-stage visual indicator system — green (safe), black (warning), red (alert) — that dynamically changes color based on live crowd count crossing configurable thresholds.
  • Triggers a flashing red alert signal with on-screen warning text when overgathering is detected, applicable to public spaces, event venues, and surveillance systems.
  • Optimised for real-time performance with frame-rate-aware processing and minimal false positives under varied lighting and occlusion conditions.
